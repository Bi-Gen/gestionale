# Gestionale

Sistema gestionale aziendale con Next.js e Supabase

## Stack Tecnologico

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Supabase (Database & Auth)

## Funzionalità

- Sistema login/autenticazione
- Anagrafica clienti (CRUD)
- Anagrafica prodotti (CRUD)
- Anagrafica fornitori (CRUD)
- Simulazione ordini

## Sviluppo

```bash
# Installa dipendenze
npm install

# Avvia server di sviluppo
npm run dev

# Build di produzione
npm run build

# Avvia produzione
npm start
```

## Deploy

Il progetto è configurato per il deploy su Vercel.
